# ⚙️ Recetas App - Backend (API REST)

Backend y API RESTful que alimenta a la aplicación "Recetas App". Se encarga de la gestión de la base de datos, la lógica de negocio (CRUD de recetas) y la validación de seguridad mediante los tokens generados por el cliente.

> **Contexto Académico:** Este proyecto fue desarrollado con fines estrictamente académicos como Trabajo Práctico Final para la asignatura Metodologia de Desarrollo Web de la carrera de Ingeniería en Sistemas. Dictada en la Universidad Abierta Interamericana (UAI)

## 🚀 Tecnologías Utilizadas

* **Entorno de Ejecución:** Node.js
* **Framework:** Express.js
* **Lenguaje:** TypeScript
* **Base de Datos:** MongoDB (MongoDB Atlas)
* **ODM:** Mongoose
* **Seguridad:** Firebase Admin SDK (Validación de tokens de sesión)
